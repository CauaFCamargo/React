# React

Projeto de estudos de React com TypeScript e Vite, desenvolvido durante os estudos na Rocketseat.

## Sobre

Aplicação criada para praticar e consolidar os fundamentos do React, explorando seus principais conceitos na prática.

## Tecnologias

- React
- TypeScript
- Vite
- CSS Modules

## Como rodar

Pré-requisitos: Node.js instalado.

```bash
# Instalar as dependências
npm install

# Rodar o projeto em modo de desenvolvimento
npm run dev
```

A aplicação ficará disponível em `http://localhost:5173`.
